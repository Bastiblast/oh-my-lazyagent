## OH-MY-LAZYAGENT
The point of this project is to keep using opencode / oh-my-openagent with cheap agent model and add Big-Brother agent to help the Sisyphus ultraworker agent when performing a hard task to give him clue and resolve its tasks.

# Stack
1. Required
- opencode
- oh-my-openagent
- flexible provider choice
2. Mandatory
- config must be portable, host in a git and easily download and install
- config must feat with many other and to collate with existing one

# Big-Brother
The idea is to add an agent like Sisyphus ultraworker that can be call by user and as subagent by Sisyphus when the main agent is encounter unresolvable problem.

# Guideline
1. Sisyphus is creating a boulder to repeat task until complet
2. if boulder get defined threshold Sisyphus make a complete report to Big-Brother to request a debug plan.
