# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-04-18

### Added

- Project structure with lazyagent directory layout
- Big-Brother agent for task tracking and context management
- Debug-plan skill for troubleshooting task execution
- Escalation hooks for detecting and handling stuck tasks
- Patch system for non-invasive OmO integration
- Install scripts for automated setup
- Configuration system with JSONC support
- Boulder agent for task planning and breakdown
- Sisyphus agent for task execution
- Validation scripts for agents and configuration
- Documentation including architecture overview and contribution guidelines
- MIT License

### Security

- Initial security review of patch system
- Validation checks for agent configurations
